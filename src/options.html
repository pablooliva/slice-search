Settings

Configuration options:

  1. Key: These characters will be used to identify which GPSE (Custom Search Engine) to use for your search.
  2. Label: Conveniently identify the intended domain of your search, used only as a reminder and has no impact on functionality.
  3. CSE Id: The id provided by GPSE. Instructions are available for how to get a GPSE id.
  4. Search narrowing terms: It is useful to add the topic of your search in this field. This term(s) will be appended automatically to your search query. This is useful in cases such as when we have a site like stackoverlow.com in our list of domains to search in GPSE, and we need to search that domain within a specific context. This context is defined by the "search narrowing terms."
